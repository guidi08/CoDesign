Descrição:

Esse projeto foi destinado ao aprimoramento pessoal da linguagem de programação Python. Criamos um software que normalizava um texto dentro de um padrão estabelecido previamente pelo professor. Esse projeto a criação de um pequeno jogo para treinar algumas habilidades mais avançadas de Python. O jogo se chamava InsperMon e era como se fosse uma releitura do Pokemon, famoso jogo que conhecemos. A única diferença é que não utilizamos computação gráfica.E simulamos as batalhas e o jogo somente com frases no prompt ,o jogador pode escrever para ou sair do jogo ,continuar ou batalhar e etc.
