Classificador de tweets

Informaçāoes básicas:

  • Curso: Ciências dos dados
  • Como? Através de conhecimentos de Probabilidade e programação
  • Quando? 2˚ semestre de 2017

Descrição:

O classificador de tweets é um programa que utiliza o teorema de bayes para classificar se uma frase é relevante ou não. O banco de dados que foram utilizados para que a maquina soubesse a probabilidade de cada palavra ser relevante ou não, foi uma quantidade suficiente de tweets sobre o tema para classificar as frases e poder ter um banco de dados de referência para a classificação de outros tweets.
